## 4. SHOPPING STANDARD

**Objectif** : afficher chaque produit sous l'onglet Shopping Google + carrousel SERP.

### Setup Google Merchant Center

1. **Vérification de domaine** : Google Search Console → ajouter `casablancaofficiel.fr`
2. **Connexion Shopify** : app Google & YouTube (gratuite) → auto-sync du flux produits
3. **Vérification du flux** : tous les produits doivent avoir :
   - Image principale (CDN)
   - Titre (max 150 caractères, mots-clés en début)
   - Description (max 5000 caractères)
   - Prix (avec devise)
   - GTIN/SKU
   - Disponibilité (in stock / out of stock)
   - Catégorie Google (Apparel & Accessories > Clothing > ...)
   - Marque (Casablanca Paris)
   - État (new)
   - Genre (Male/Female/Unisex)
   - Groupe d'âge (Adult)

### Optimisation des titres pour Shopping

Pattern recommandé : **Marque + Type + Modèle + Couleur + Matière + Taille (optionnel)**

Exemple :
```
❌ Tennis Club Icon T-Shirt
✅ Casablanca Paris Tennis Club Icon T-Shirt Homme Coton Crème
```

## 8. Setup tracking & conversions (CRUCIAL)

### Conversions à configurer (Google Ads → Outils → Conversions)

| Conversion | Type | Valeur | Action |
|---|---|---|---|
| **Achat** | Conversion principale | Dynamique (panier total) | `Purchase` event |
| **Add to Cart** | Secondaire | Statique 5 € | `AddToCart` event |
| **Begin Checkout** | Secondaire | Statique 10 € | `InitiateCheckout` event |
| **Lead/Newsletter** | Secondaire | Statique 2 € | Submit newsletter form |
| **View Item** | Micro | Statique 0,50 € | `ViewItem` event |

### Implémentation

1. **Google Tag Manager** installé sur Shopify (via app gratuite)
2. **Google Ads Tag** + Enhanced Conversions activées (envoi email hashé après achat pour matching)
3. **Server-side tracking** (recommandé) via app Stape ou Shopify Server Side
4. **Consent Mode v2** activé pour respect RGPD (notre bandeau cookies est déjà compatible)
